# Face Mask Detection

This project is a **real-time face mask detection system** that detects whether a person is wearing a mask using a webcam.

## Technologies Used
- Python
- OpenCV
- TensorFlow / Keras
- MobileNetV2

## Features
- Detects faces in real-time using a webcam
- Classifies as **Mask** or **No Mask**
- Lightweight and fast model for quick deployment

## Files in Repository
- `detect_mask.py` → Python script for real-time detection
- `README.md` → Project description
